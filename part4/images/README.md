# Images Directory

This directory should contain the following image files:

## Required Images

### logo.png
- **Purpose**: Application logo displayed in the header
- **Recommended Size**: 200x60px or similar aspect ratio
- **Format**: PNG with transparency
- **Description**: Should contain the HBnB logo (red seahorse icon + "hbnb" text as shown in the design)

### icon.png
- **Purpose**: Favicon displayed in browser tab
- **Recommended Size**: 32x32px or 16x16px
- **Format**: PNG or ICO
- **Description**: Small icon version of the logo

## Placeholder Images

If you don't have the actual logo images yet, you can:
1. Use the SVG placeholders provided (logo.svg, icon.svg)
2. Create simple placeholder images
3. Use online logo generators

## Note

The HTML files reference these images. Make sure to add the actual logo.png and icon.png files to this directory for the application to display correctly.
